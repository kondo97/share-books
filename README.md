## 使用技術
- Nuxt 2.15.8
- Vuetify 2.5.5
- firebase 8.9.1
- algoliasearch 4.10.5

## 機能
- ユーザー登録、ログイン機能
- 投稿機能
　オススメの本を紹介する記事の投稿
- 検索機能
　タグ検索、全文検索
- いいね機能
- コメント機能
- フォロー機能
- ページネーション機能
- SNSシェア機能
